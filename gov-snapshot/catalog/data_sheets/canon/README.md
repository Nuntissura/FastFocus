# Canon data sheets (digital still)

- `FF - gov/catalog/data_sheets/canon/model-lists/`

Current scaffolded index (generated):
- `FF - gov/catalog/data_sheets/canon/model-lists/canon_digital_still_models_scaffolded.tsv`

- `FF - gov/catalog/scripts/scaffold_canon_digital_still_datasheets.mjs`

- `FF - gov/catalog/scripts/enrich_canon_from_camera_museum.mjs`

Import readiness report (generated by importer dry-run):
- `FF - gov/catalog/data_sheets/canon/model-lists/canon_datasheet_import_readiness.tsv`

Notes:
- This brand folder is scoped to Canon **digital still cameras** only. Analogue/film cameras will use a separate template later.
